# TwinTalk - Voice-Actuated Twin Rotor Simulation Layer

This repository contains the 3D kinematic simulation and safety-gate orchestration layer for the Twin Rotor V1.0 platform (ME 325 Project).

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Launch the server: Double-click `Run_TwinTalk_Sim.bat` or run `python app.py`
3. Open the MeshCat visualizer link printed in the terminal (`http://127.0.0.1:7000/static/`).
